What ?

This application show you in real time the usage of the BiXi system in Montreal. Each bar represents a BiXi station. The green part is the amount of bikes available, the yellow one the amount of missing bikes. If the bar is red it means that the station is currently locked.

This map is refreshed every minute, there is a counter on the top left part of the map to display the time remaining to the next refresh. You can move around using the arrows on your keyboard. Page up and Page down change the camera's view angle. A and Z let you rotate around the map.

Focusing a bar with your mouse cursor will display more information about the station in the upper left corner.

Why ?

For several years I was (and still am ) a game developer, I spent some long hours writing code for games and my job right now evolve around gaming (You can see more about my professional side here).

Point is, I see more and more people asking for web applications and I was eager to know what is the current state of WebGL and is it usable right now ? So I decided to write this little application to do some data visualisation.

As for the data, Francis Gosselin from f. & co was around and told me about the BiXi system in Montreal. The datasource was available and produces some real time data. Exactly what I was looking for.

How ?

I've used several tools to create this web page. For the design my starting point was Twitter Bootstrap, despite the fact that I ended doing a custom page set up myself. For the 3D I use WebGL, specifically Three.js. And finally the back end was written using Django.

The service I used to poll the data is located here. The map is draw using data from Open Street Map. I will certainly enhance this application as I've got several ideas of some fun stuff we could do here.
